# 🎓 Course Management System

A modern and responsive Course Management System developed to simplify course organization and student learning management. This project provides an intuitive interface for students to explore courses, access learning resources, and manage their academic activities efficiently.

## 📌 Overview

The Course Management System is designed to streamline the management of courses and educational content. It offers a user-friendly dashboard where students can browse available courses, view course details, and track their learning journey.

This project focuses on creating a clean and responsive frontend experience using modern web technologies.

## ✨ Features

* Responsive and user-friendly interface
* Modern dashboard design
* Course listing and management
* Detailed course information pages
* Easy navigation between sections
* Mobile-friendly layout
* Clean and interactive UI components
* Organized project structure

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/kiruthiga-07/Course-Management-System.git
```

### Navigate to the Project Directory

```bash
cd Course-Management-System
```

### Run the Project

Open the `index.html` file in your browser.

Alternatively, use the VS Code Live Server extension:

```bash
Right Click → Open with Live Server
```
